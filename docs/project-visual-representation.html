4) Management Integrations

JIRA
  • Create issue per pipeline execution summary
  • Env: JIRA_BASE_URL, JIRA_EMAIL, JIRA_API_TOKEN, JIRA_PROJECT_KEY
Confluence
  • Create execution page with pipeline output
  • Env: CONFLUENCE_BASE_URL, CONFLUENCE_SPACE_KEY, credentials
Xray
  • Import test execution payload scaffold
  • Env: XRAY_BASE_URL, XRAY_CLIENT_ID, XRAY_CLIENT_SECRET

Publishing is optional and controlled via PIPELINE_PUBLISH_MGMT=true.
